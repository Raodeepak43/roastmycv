**How to ask AI to improve your resume** is one of Google's top People Also Ask questions for resume searches (live SERP, July 2026). Most freshers paste their CV into ChatGPT with a vague prompt like "make it better" — and get generic fluff back.

Here's how to prompt AI properly — or skip prompts entirely with a [free resume roast](/).

## The Prompt That Actually Works

```
You are a brutal Indian tech recruiter. Read my resume below.

1. Score it /10 for [TARGET ROLE] at [COMPANY TYPE]
2. Quote 3 weakest lines verbatim and explain why they fail
3. List 5 specific fixes (not generic "add keywords")
4. Flag ATS risks (columns, graphics, missing metrics)
5. Rewrite my summary in 2 lines max

Resume:
[PASTE FULL TEXT]
```

**Problem:** ChatGPT forgets context, won't read PDF layout well, and gives polite answers unless you push hard.

**Faster path:** [MyCVRoast](/) — upload PDF, pick Hinglish + intensity, get structured output in ~30 seconds.

## Step-by-Step: AI Resume Improvement Workflow

### 1. Extract Clean Text

- Export PDF to text or use a tool that parses layout
- Remove headers/footers that break ATS
- One column, standard fonts

### 2. Define Target Role

AI needs a target. "Improve my resume" is useless. Try:

- "SDE-1 at product startup, Bangalore"
- "Campus placement — TCS/Infosys mass hiring"
- "Data analyst intern, 0 experience"

### 3. Run Roast First, Rewrite Second

Order matters:

1. **Roast** — find what's broken ([free roast](/))
2. **Rewrite bullets** — [Bullet Rewriter tool](/blog/resume-bullet-rewriter-ai)
3. **JD match** — compare to job description ([JD Match](/blog/jd-match-ats-score-tool))
4. **Final roast** — confirm fixes worked

### 4. Use India-Specific Context

Tell AI you're an Indian fresher if relevant:

- CGPA vs percentage
- Internship vs "work experience"
- One-page rule for campus drives
- Hinglish feedback preference

[MyCVRoast Hinglish roast](/tools/resume-roast-in-hinglish) handles this by default.

## Common AI Resume Mistakes

| Mistake | Fix |
|---------|-----|
| "Make my resume ATS friendly" (no JD) | Paste the actual job description |
| Accepting invented metrics | Only use real numbers you can defend in interview |
| One-shot prompt | Iterate — roast → fix → roast again |
| Ignoring format | Use [ATS resume builder](/resume-builder) first |
