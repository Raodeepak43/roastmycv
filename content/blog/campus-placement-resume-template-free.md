## What a Good Template Looks Like

```
┌─────────────────────────────────────────┐
│  YOUR NAME                              │
│  phone | email | LinkedIn | GitHub      │
│  City, State                            │
├─────────────────────────────────────────┤
│  SUMMARY                                │
│  2 lines: role target + best proof      │
├─────────────────────────────────────────┤
│  EDUCATION                              │
│  B.Tech CSE | College | CGPA 8.2 | 2026 │
├─────────────────────────────────────────┤
│  TECHNICAL SKILLS                       │
│  Languages: ... | Frameworks: ...       │
├─────────────────────────────────────────┤
│  PROJECTS                               │
│  • Project name — stack — outcome       │
│  • Project name — stack — outcome       │
├─────────────────────────────────────────┤
│  INTERNSHIPS                            │
│  • Company | duration | 2 bullets      │
├─────────────────────────────────────────┤
│  ACHIEVEMENTS                           │
│  • Hackathon rank / certification       │
└─────────────────────────────────────────┘
        ONE PAGE — NO COLUMNS
```
