**Off-campus placement** mein college TPO shortcut nahi hota — resume khud sell karni padti hai on Naukri, LinkedIn, company careers pages, and referrals.

## Campus vs off-campus resume difference

| Campus | Off-campus |
|--------|------------|
| Batch competition | You vs national pool |
| Standardized 1-page | Needs portfolio links |
| Company comes to you | You chase roles |

## Tip 1: GitHub + live demo links

CS/IT roles — code proof mandatory. README with setup steps.

## Tip 2: Personalized summary per apply

*"Applying for [Company] [Role] — BTech 2026 with X project using Y stack."*

## Tip 3: LinkedIn match

Resume keywords = LinkedIn headline skills. Recruiters cross-check.

## Tip 4: PDF name professional

`FirstName-LastName-Role.pdf` not `resume_final_v9.pdf`.

## Tip 5: Track applications

50+ applies with same generic CV = [no calls problem](/blog/no-interview-calls-after-50-applications).

## Tip 6: Referral line in email

Short cover note + resume — not empty apply button only.
