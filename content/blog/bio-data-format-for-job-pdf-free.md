## Bio data vs resume vs CV (India)

| Document | Typical use | Personal details |
|----------|-------------|------------------|
| **Bio data / biodata** | Government, PSU, banking, teaching, some private forms | Sometimes DOB, father's name, marital status |
| **Resume** | Private companies, startups, IT campus, BPO | Minimal — contact + professional only |
| **CV** | Academia, research abroad | Publications, longer format |

Confused which to send? If the form says **"biodata"** or **"bio data"** — use this format. If Naukri/LinkedIn/startup — use [ATS resume](/blog/ats-friendly-resume).

## Free bio data format for job (copy structure)

Use this in Word or Google Docs → Export PDF.

```
BIO DATA

Name: [Full name]
Date of Birth: [Only if form requires]
Father's Name: [Only if form requires]
Address: [City, State — full address if form asks]
Phone: [10-digit mobile]
Email: [Optional]

OBJECTIVE
[One line — specific job title, e.g. Clerk, Teaching assistant, Bank PO trainee]

EDUCATION
• [Degree/Board] — [Institution] — [Year] — [% or grade]
• [10th/12th] — [Board] — [Year] — [%]

EXPERIENCE / TRAINING
• [Role] — [Organisation] — [Dates] — [1 line duty]

SKILLS
• [Languages, computer, relevant certificates]

DECLARATION
I hereby declare that the above information is true to the best of my knowledge.

Place: _________    Date: _________    Signature: _________
```

**Do not** add father's name, religion, or marital status for IT/MNC resumes unless the employer form explicitly requires it.

## Government vs private company bio data

**Government / PSU / banking:**
- Follow notification PDF exactly — extra columns, attestation, photo size
- Bio data often mandatory; resume alone may be rejected

**Private BPO / retail / IT:**
- Usually want **resume** not marriage-style biodata
- [12th pass resume](/blog/12th-pass-resume-kaise-banaye) · [10th pass resume](/blog/10th-pass-resume-kaise-banaye)

## Common mistakes

- Using marriage biodata template for IT jobs — too personal, wrong sections
- Scanned handwritten page — OCR fails online applications
- "Bio data" title but resume content with no declaration block (when form expects declaration)
- Fake experience — verification in PSU and banking

## Freshers — bio data or resume?

| Situation | Send |
|-----------|------|
| TCS / Infosys campus | ATS resume |
| Bank clerk form "attach biodata" | Bio data format above |
