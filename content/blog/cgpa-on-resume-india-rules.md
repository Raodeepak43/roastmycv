**CGPA on resume India** — show it or hide it? Freshers debate this every placement season. Hide a 9.2 and you look suspicious. Flash a 6.8 without context and TCS auto-filters you.

Here's the recruiter logic for 2026 campus and off-campus hiring.

**[Check if rest of CV saves you →](/tools/ats-score-checker-for-freshers)**

## The Short Answer

| Situation | Put CGPA? |
|-----------|-----------|
| Fresher, campus drive | **Yes** — always in Education |
| Fresher, CGPA ≥ 7.0 | **Yes** — prominently |
| Fresher, CGPA 6.0–6.9 | **Yes** — pair with strong projects |
| Fresher, CGPA < 6.0 | **Yes** — but lead with projects/internships |
| 2+ years experience | Optional — moves to bottom |
| MBA fresher | **Yes** — often stricter cutoffs |

## Where to Put CGPA (Format)

**Correct:**
```
B.Tech Computer Science
XYZ Institute of Technology | CGPA: 8.4/10 | Expected 2026
```

**Wrong:**
- Footer in 8pt font
- "Academic performance: satisfactory"
- Only in cover letter, not resume
- Mixed with skills: "CGPA 8.4, Python, Java"

## CGPA Cutoffs (Typical — Verify Per Drive)

| Company type | Common floor |
|--------------|--------------|
| TCS / Infosys / Wipro | 6.0–6.5+ (varies by college tier) |
| Product startups | Often flexible if projects strong |
| Core engineering | May weight CGPA higher |
| Service + MBA | 7.0+ common |

## When Hiding CGPA Backfires

Recruiters assume:
- You have something to hide
- You didn't meet eligibility
- Resume is incomplete

If CGPA is weak, **compensate visibly:**

```
PROJECTS
• E-commerce API — Node.js — 2K users, deployed on AWS
• ML classifier — Python — 94% accuracy, GitHub 50+ stars
```

Then [roast your CV](/) — AI will tell you if projects are strong enough to offset CGPA.

## CGPA vs Percentage

- **CGPA scale:** State clearly (/10, /4, /9.5 — whatever your college uses)
- **Don't fake convert** 6.5 CGPA to 85% unless that's official
- **Backlogs:** If cleared, show CGPA + "No active backlogs" if true

## CGPA on Naukri vs Resume PDF

- Naukri profile has separate education fields — **fill them**
- PDF resume must match Naukri data (recruiters cross-check)

## Engineering Students

CGPA + GitHub + projects triangle:

| CGPA | Strategy |
|------|----------|
| High (8.5+) | Lead education, add projects for depth |
| Mid (7–8.4) | Balance projects and CGPA |
| Low (<7) | Projects first, open source proof, internships |
